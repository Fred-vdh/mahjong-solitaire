# Mahjong-solitaire 1.0

Un jeu de Mahjong Solitaire élégant développé en Python avec Pygame, offrant une expérience visuelle riche et des fonctionnalités avancées.

## ✨ Fonctionnalités

- **Esthétique Soignée :** Tuiles avec un aspect "ivoire vieilli" et patiné pour une ambiance authentique.
- **Animations Fluides :** 
  - Transitions de niveaux tuile par tuile.
  - Plusieurs animations de victoire dynamiques (dont l'effet "Fontaine").
  - Animations de mélange (shuffle) et d'annulation (undo) avec délais naturels.
- **Éditeur de Niveaux :** Un outil complet pour créer et modifier vos propres dispositions de tuiles.
- **Statistiques Détaillées :** Suivi de vos performances par niveau (temps, scores, mélanges).
- **Ambiance Sonore :** Musiques relaxantes et effets sonores de clics.
- **Système d'Indices :** Aide automatique ou manuelle pour trouver des combinaisons.

## 🚀 Installation et Lancement

### Pour les joueurs (Windows)
Vous pouvez trouver l'exécutable dans le dossier `dist/` (si disponible) ou le compiler vous-même.

### Pour les développeurs
1. Assurez-vous d'avoir Python installé.
2. Installez les dépendances :
   ```bash
   pip install pygame numpy
   ```
3. Lancez le jeu :
   ```bash
   python mahjong_game.py
   ```
4. Lancez l'éditeur :
   ```bash
   python mahjong_editor.py
   ```

## 🛠️ Technologies utilisées
- **Python 3**
- **Pygame** : Moteur de rendu 2D.
- **NumPy** : Calculs mathématiques pour les animations.
- **PyInstaller** : Pour la création de l'exécutable.
